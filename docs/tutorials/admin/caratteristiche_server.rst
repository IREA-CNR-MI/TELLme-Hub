====================
Server requirements
====================

.. toctree::
   :maxdepth: 3


GET-IT viene fornito attraverso macchina virtuale VMware sotto licenza open (GPL v3.0) e può essere installato su un qualsiasi server (Linux, Window, MAC) su cui sia installato il client VMware.

In caso di esigenze specifiche, la macchina virtuale può essere fornita con altri formati standard.

Lo starter kit non necessita un server particolarmente performante, di seguito indichiamo le caratteristiche consigliate secondo la nostra esperienza (non vincolanti):

* RAM: 6 GB minimo, consigliato 8 GB
* PROCESSORI:  1 processore 4 core
* HDisk capacity: la macchina virtuale occupa 30 GB
* hardware raccomandato 64-bit 


Queste caratteristiche possono essere aumentate per migliorare le prestazioni o in caso di grosse quantità di dati inseriti nel portale.
